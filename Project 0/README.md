## simple example tRCP with ExpressJS (TS/JS)

### Runs the tRPC server ts

```
npm run server-ts
```

### Runs the tRPC server js

```
npm run server-js
```

### Runs client ts

```
npm run client-ts
```

### Runs client js

```
npm run client-js
```

### Runs client express ts

```
npm run client-express-ts
```

### Runs client express js

```
npm run client-express-js
```
